Calculator

For incorporating a negative value as first operand, first enter the value and then use plusMinus operator and then hit equals.

IF you get an "undefined" result (while using operations like power, square, square-root, inverse), hit the operator again or hit C/CE/F5 and do it again, you'll get the proper result.

If you hit multiple operators one after the other, the currently displayed operator will get replaced by the latest operator.
